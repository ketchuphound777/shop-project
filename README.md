# shop-project

Welding, Parts Organization, and Database Entry
This folder documents my work welding shelves, organizing inventory parts, and using Excel to prepare data for import into a company’s database. The goal of this project was to improve storage efficiency, create durable shelving, and streamline part tracking using structured spreadsheets.
